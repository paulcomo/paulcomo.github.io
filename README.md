# Paul Como

Source for my personal technical blog, published with GitHub Pages at
[paulcomo.github.io](https://paulcomo.github.io).

The site focuses on homelab automation, infrastructure as code, IT operations,
and practical notes from projects I am building or maintaining.

## Stack

- Jekyll
- Chirpy theme
- GitHub Pages
- GitHub Actions deployment

## Local Development

```bash
bundle install
bundle exec jekyll serve
```
